## General info
A simple tkinter app to download youtube videos
	
## Technologies
Project is created with:
* pytube==10.4.1
	
## Setup
To run this project,

```
$ git clone https://github.com/baburajr/youtube-downloader.git
$ cd youtube downloader
$ pip install -r requirements.txt
$ python3 youtube.py
```
